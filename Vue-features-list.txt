What I can do with VUE:

- Show/Hide single elements on click (v-if)
- Show/Hide templates(groups of elements) on click
- Show/Hide elements on click(display="none")(v-show)
------------------
- Output lists (v-for)
- Output lists with index (current_item, index) (v-for)
- Output lists using HTML5 template tag (v-for)
- Output arrays holding objects (nested v-for)
- Output a range of integers (v-for)
------------------
- Create multiple app instances
------------------
- Interact with HTML elements in JS ($refs)
-------------------
- Create reusable components with their own templates (Vue.component())
- Register components globally or locally
- Pass data from the parent to the child components (props)
--------------------
- Use slots to pass multiple elements to child component
- Use named slots 
--------------------
- Load different content on page on click (dynamic components)
---------------------
- bind form data to an object (v-model)
- use modifiers on input (lazy, number, trim)
- use custom components as controls
